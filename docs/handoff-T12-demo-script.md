## 任务目标

编写一份 5 分钟的劳动争议咨询对话脚本，用于 Demo 演示。脚本需自然触发 AI 侧边栏的三类卡片：法规引用、合同条款建议、风险提示。

## Demo 节奏

```
0:00-0:30  开场：律师与客户寒暄，AI 侧边栏显示「正在听...」
0:30-1:30  客户描述问题：入职后未签劳动合同
1:30-2:00  第一个 wow：AI 弹出《劳动合同法》第82条（法规卡片）
2:00-3:30  深入：公司拖欠工资 + 拒绝补签
3:30-4:00  第二个 wow：AI 弹出解除劳动合同协议模板（合同卡片）+ 风险提示
4:00-5:00  律师总结 + AI 一键输出完整分析报告
```

## 对话脚本

```
律师：您好，请坐。今天过来是想咨询什么问题？
客户：是这样的，我去年3月份入职了一家科技公司，做软件开发。
律师：嗯，入职的时候签劳动合同了吗？
客户：没有。当时HR说先干着，合同过两个月补签。
律师：现在补签了吗？
客户：没有，到现在都没签。而且上个月开始，公司说经营困难，工资只发了一半。
律师：您现在还在职吗？
客户：还在，但我已经不想干了。就想知道我能要多少钱。
律师：您一个月工资多少？
客户：税前一万五。
律师：好的。您这个情况我理解了——入职一年多没签合同，现在公司还拖欠工资。
      根据劳动合同法的规定，您这个情况对您还是比较有利的。
      [此时 AI 侧边栏应弹出：劳动合同法第82条 + 第85条]
客户：那我能拿到多少钱？
律师：主要有几笔。第一，没签合同期间的二倍工资差额。第二，拖欠工资的补发。
      第三，如果您以此为由解除合同，还可能有经济补偿。
      [此时 AI 侧边栏应弹出：赔偿计算 + 解除劳动合同协议模板]
客户：那我怎么跟公司谈？
律师：我建议先书面通知公司，明确要求补签合同、补发工资。
      如果公司不理，就可以申请劳动仲裁。
      [此时 AI 侧边栏应弹出：风险提示 + 仲裁流程指引]
客户：好的，那就按您说的办。
律师：行，我把刚才说的情况整理一下，给您出一份初步的分析报告。
      [手动触发：AI 输出完整分析报告 —— 法规依据 + 赔偿估算 + 行动建议]
```
